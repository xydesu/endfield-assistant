快速開始

如何使用

只需幾個步驟，即可讓機器人每天自動幫你完成簽到。

1

邀請機器人加入你的伺服器

點擊上方「加入機器人」按鈕，將 Endfield Assistant 授權加入你的 Discord 伺服器。

2

執行 /bind 綁定帳號

在任何機器人可見的頻道輸入 /bind，按照互動式嵌入訊息的步驟操作。前往簽到網站，在瀏覽器開發者主控台（F12 → Console）執行提供的腳本，取得你的 Session 憑證。

3

貼上憑證完成綁定

將腳本輸出的 JSON 貼入點擊「輸入 Config」後彈出的表單。你的憑證將以 AES-256-CBC 加密儲存。

4

設定排程（選用）

使用 /schedule HH:mm 設定每日自動簽到時間。也可以隨時使用 /signin 手動觸發簽到。
